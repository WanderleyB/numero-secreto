# 🎮 Número Secreto

Projeto simples desenvolvido para **praticar conceitos de Git e GitHub** durante o processo de aprendizagem de versionamento de código.

O objetivo deste repositório é compreender na prática como funcionam os principais comandos do Git, organização de commits e publicação de projetos no GitHub.

---

## 📚 Sobre o Projeto

Este projeto consiste em um pequeno jogo em JavaScript onde o usuário deve adivinhar um **número secreto gerado aleatoriamente pelo sistema**.

A cada tentativa o sistema informa se o número digitado é **maior ou menor** que o número correto, até que o jogador descubra o valor secreto.

---

## 🎯 Objetivos de Aprendizado

Este projeto foi criado com foco em estudar:
